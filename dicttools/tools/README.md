# ツール群

このディレクトリには、ひともじのツール類を置いています。

## 詳細

- dictview.cpp
  バイナリ辞書えつ覧ツール
  パラメタとして読みを指定すると、辞書ファイルから展開/表示するツール

- dictedit.cpp(未実装)
  辞書メンテツール。
  独自の読みや、文字を追加/修正/さく除するCLIツール
  基本的には、add/edit/deleteの3コマンドが使用できる。
  このツールの出力先は、バイナリ辞書ではなくSQLite3のDBとなる。
  辞書への反映には、../buildのツール群が必要となる

## 移行方針

- `../dictview.cpp` は、将来的にこのディレクトリへ移動する。
- `dictedit.cpp` を実装する場合は、このディレクトリへ置く。
- 辞書生成そのものに使うツールは `../build/` または各生成フェーズへ置く。
